# 科创板 / 北交所小资金优势趋势筛选（第一轮）

## 1. 筛选口径

依据 `docs/Investment Analysis Framework for Small-Capital Advantages.md`，本轮优先寻找：基本面改善、存在预期差、月线趋势开始或中段确认、估值没有过度拥挤、行业处于长期景气方向的公司。

本轮使用的本地数据源：

1. `data/processed/a_share_watchlist_quality_tiers.csv`：A股观察池质量分层与财务辅助指标。
2. `data/processed/a_share_focus_watchlist_l1_l2_valuation.csv`：L1/L2 当前估值分档。
3. `data/interim/a_share_authoritative_review_notes_batch-*.csv`：年报、一季报和机构研报摘要。

限制：实时月线行情接口在本环境中未稳定返回，因此 `monthly_trend_status` 暂标为 `needs_live_kline_confirmation`。下面名单不是买入清单，而是下一步应优先核对月线和研报上修的研究池。

## 2. 优先研究候选

艾力斯（688578），心脉医疗（688016），特宝生物（688278），时代电气（688187），睿创微纳（688002），思特威（688213）

共同特征：已有质量分层不低，2026Q1 收入和利润均有较好增长，现有估值分档为“较低估”，并且行业方向属于创新药/高端医疗器械/红外感知/CIS/轨交电力电子等长期景气或国产替代方向。

## 3. 条件候选

金山办公（688111），惠泰医疗（688617），柏楚电子（688188），联影医疗（688271），达梦数据（688692），奕瑞科技（688301），安集科技（688019），百济神州（688235），键凯科技（688356）

这些公司业务质量较强，但当前估值多为“中性”，或者利润中存在一次性收益、成长持续性、客户验证、采购周期等问题。只有在月线温和上行且研报预期继续上修时，才适合进入准买入池。

## 4. 北交所待核验候选

中科仪（920186），同惠电子（920509），惠同新材（920751），锦波生物（920982）

北交所公司需要额外严格：流动性、研报覆盖、成交额、估值数据和披露质量都必须二次核验。暂不把它们放入优先买入候选。

## 5. 暂不追高 / 拥挤剔除

澜起科技（688008），中微公司（688012），海光信息（688041），寒武纪（688256），中芯国际（688981），华峰测控（688200），华海清科（688120），拓荆科技（688072），强一股份（688809），源杰科技（688498），绿的谐波（688017），芯源微（688037），中科飞测（688361），中船特气（688146），盛科通信（688702）

这些公司多属于半导体、AI算力、半导体设备/材料或机器人等高景气方向，业务可能很好，但当前估值已经明显提前反映远期增长，不符合“当前不处于极度高估”的筛选条件。

## 6. 下一步核验清单

1. 对优先研究候选逐个补月线：6月、10月、12月均线，近3/6/12个月收益，是否处于上升初段或中段，是否出现末端加速。
2. 对优先研究候选逐个补研报：最近3-6个月盈利预测是否持续上修，目标价上修是否依赖主业兑现，而不是单纯拔估值。
3. 对北交所候选补成交额和流动性：如果日均成交额过低，即使趋势和基本面较好，也只能战术观察。
4. 对中性估值候选补PEG和成熟期利润率：只有2-3年base case收益/ bear case风险足够高，才允许进入准买入池。
