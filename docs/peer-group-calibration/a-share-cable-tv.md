# 1. A-Share Cable TV Peer-Group Review
## 1.1 Purpose
This note records the company-by-company review for `文化传媒-广播电视-有线电视`.

This group contains provincial cable-network operators, IPTV/OTT platforms, state media capital platforms, and video-transmission equipment suppliers. Traditional cable TV is structurally pressured, so the review retains only network franchises or new-media platforms with real user access, regional network control, or content/platform optionality.
## 1.2 Retention Standard
1. Directly retain companies with national or high-quality new-media/IPTV platform control, content and user access, and stronger platform optionality than ordinary provincial cable networks.
2. Boundary-retain regional cable networks only when the geography, broadband/IDC extension, government network role, or local monopoly is materially stronger.
3. Reject weaker provincial cable operators and equipment suppliers where the core cable subscriber base is declining and no scalable content or data platform is evident.
