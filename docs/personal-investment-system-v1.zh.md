# 个人投资体系 v1.2

> 本文是个人投资规则手册，不构成投资建议。所有数值阈值都是初始规则，应通过交易记录和复盘持续迭代。
>
> 分工：本文件是账户级总规则（组合结构、策略分类、买卖纪律，覆盖 A 股/港股/美股）；A 股市场的操作化执行（全市场质量筛选、估值排雷、每日量价扫描、持仓卖出扫描）由 `docs/000_Ashare_workflow.md` 承担。该工作流中复用的组合暴露与回撤阈值以本文件 §2/§13 为源，修改任一处必须同步另一处。

## 1. 核心原则

本体系的定位是：

```text
价值投资底层 + 产业趋势增强 + 右侧交易执行 + 仓位风险预算。
```

基本执行原则：

```text
研究决定买什么。
右侧信号决定何时买。
风险预算决定买多少。
thesis 是否变化决定加仓、减仓或清仓。
成本价不能决定任何持仓动作。
```

目标不是永远满仓，也不是永远长期持有，而是在控制永久性亏损和杠杆风险的前提下，集中持有少数高质量、高景气、高赔率或深度低估资产。

最高优先级规则：

```text
如果买入 thesis 被证伪，或者财务质量、治理结构、行业逻辑出现重大破坏，可以并且必须清仓。
```

观察池最高优先级规则：

```text
观察池必须严进宽出。
宁可错过普通机会，也不要把低壁垒、低确定性、估值不够便宜的公司放进长期观察池。
```

“值得研究一次”和“值得长期观察”必须分开。很多公司可以被分析、理解、排除，但不应因为业务有增长、股价下跌、题材顺眼或短期有反弹可能，就进入观察池。

## 2. 账户结构

### 2.1 持仓层级

| 层级 | 单只仓位 | 数量 | 合计上限 | 作用 |
| --- | ---: | ---: | ---: | --- |
| 低估优质仓（L1/L2×低估/较低估+右侧信号，v20） | <=10% | 4-8 只 | <=60% | 账户主要收益来源 |
| 突破/事件仓（核心池信号或L3×中性事件通道，v20） | <=5% | 3-5 只 | <=15%（战术合计） | 右侧动量、事件动量、异动响应 |
| L4 低估极小仓 | <=2.5% | 0-2 只 | 计入战术合计 | 深度低估期权型 |
| 现金 / 备用资金 | - | - | 0%-20% | 等待机会、控制回撤、杠杆安全垫 |

仓位基准为总资产（净资产+融资负债）；风险预算（单笔1.5%、回撤梯）按净资产口径。原核心持仓 15%-25% 与准核心 4%-10% 档自 v20 暂停使用（存量持仓不强制调整，按减仓复核处理），待体系连续稳定运行两个季度后复核启用；一笔建仓即可，前置为 §9.1 清单+割肉价+1.5% 单笔风险校验。

硬性限制：

| 规则 | 上限 |
| --- | ---: |
| 单只股票常规上限（v20新开仓） | 10% |
| 单只股票极限上限（存量绝对红线） | 30% |
| 同一产业链合计暴露 | 40% |
| 同一大行业合计暴露 | 60% |
| 战术持仓合计 | 15% |
| 烟蒂 / 困境类持仓合计 | 15% |

核心仓接近 30% 必须同时满足：

1. 公司财务健康。
2. 行业景气度向上。
3. thesis 有财报、订单、公告、产业数据支持。
4. bear case 已明确写出。
5. 下跌 30% 不会导致被动爆仓或心理崩溃。
6. 最新财报、订单或产业证据仍支持 thesis。

### 2.2 观察池层级

| 层级 | 数量 | 含义 |
| --- | ---: | --- |
| 全量观察池 | 30-80 只 | 值得长期跟踪 |
| 重点观察池 | 10-20 只 | 近一个月最值得关注 |
| 准买入池 | 3-8 只 | 研究完成，等待触发 |
| 实际持仓 | 8-15 只 | 当前账户持仓 |
| 核心持仓 | 2-3 只 | 账户主要驱动 |

不要为了观察而买很多 1% 小仓位。观察主要通过观察池完成；战术持仓只用于已经出现触发信号的公司。

观察池入选必须至少满足以下条件之一：

1. 公司具有强护城河，资金充足的新进入者也难以在短期内复制、替代或战胜。
2. 公司处于关键产业瓶颈，供给稀缺、认证周期长、扩产难、客户替换成本高。
3. 公司拥有稀缺资源、牌照、成本曲线优势、网络效应、强品牌心智或不可轻易复制的渠道能力。
4. 公司估值进入深度低估，并且资产真实、现金流可验证、存在明确催化剂。
5. 公司是落难白马，但核心业务和护城河未被永久破坏，且已有右侧修复迹象。

以下情况默认不进入观察池：

1. 行业进入门槛低，新玩家只要有资金就可以较容易复制产品、渠道或产能。
2. 高利润会迅速吸引新玩家进入，导致利润率和增长难以长期维持。
3. 增长主要靠营销投放、渠道铺货、价格周期、出口景气或短期费用资本化，而不是长期竞争优势。
4. 公司只是“还不错”，但没有明显强于同业的不可替代优势。
5. PE 约 20-25 倍，但公司没有强护城河、强现金流、强复利能力或明确低估催化。
6. 只是股价跌多了、技术面可能反弹、短期业绩不错，但长期竞争力不足。

如果入选理由需要用很多假设才能成立，默认不入选。

## 3. 市场状态与总仓位

### 3.1 市场状态判断

按主要投资市场分别判断，例如 A 股看沪深 300，港股看恒生指数，美股看标普 500 或纳斯达克 100。

| 状态 | 判断标准 |
| --- | --- |
| 强势市 | 指数在 250 日线上方，250 日线向上，市场宽度改善，成交额放大，有清晰主线 |
| 弱势市 | 指数在 250 日线下方，250 日线向下，市场宽度弱，无持续主线 |
| 震荡市 | 既不满足强势市，也不满足弱势市 |

市场状态至少维持约 20 个交易日后再确认。市场状态只控制总仓位和进攻性，不直接强制清掉优秀个股。

### 3.2 总仓位区间

| 市场状态 | 股票总仓位 | 优先策略 |
| --- | ---: | --- |
| 强势市 | 75%-100% | 产业链爆发、成长龙头、高景气核心 |
| 震荡市 | 45%-75% | GARP、低估催化、股东回报、局部突破 |
| 弱势市 | 15%-45% | 现金流复利、垄断资源、高分红、深度低估 |

## 4. 杠杆规则

体系运行第一年禁用杠杆。满一年后，只有在真实记录证明回撤和纪律可控时，才允许使用。

### 4.1 杠杆上限

| 市场状态 | 总杠杆 |
| --- | ---: |
| 弱势市 | 0.5x-0.9x |
| 震荡市 | 0.8x-1.1x |
| 结构性牛市 | 1.0x-1.2x |
| 强牛市 + 高确定性机会 | 1.2x-1.3x |

集中持仓的个人账户长期杠杆不得超过 1.3x。

### 4.2 可加杠杆资产

只有以下资产可以考虑杠杆：

1. 流动性高。
2. 财务稳健。
3. 估值有安全边际。
4. 不依赖单一订单、单一客户、单一政策或传闻。
5. 没有明显财务红旗。
6. 压力测试显示组合下跌 25%-30% 也不会被迫平仓。

禁止对以下资产加杠杆：

1. 纯题材股。
2. 单一订单、单一客户、单一验证驱动的公司。
3. 还没有验证的高估值成长股。
4. 财务复杂的烟蒂股。
5. 周期顶部的周期股。
6. 短期已经暴涨数倍的股票。
7. 小盘低流动性股票。

## 5. 选股策略分类

每只股票买入前必须有一个主要策略标签。卖出规则跟随原标签，除非完成完整重估。

| 代码 | 策略 | 最高层级 | 核心问题 |
| --- | --- | --- | --- |
| A | DCF / 现金流复利型 | 核心 | 不依赖乐观假设，能否长期复利？ |
| B | 烟蒂深度低估型 | 准核心 | 是否便宜、资产真实、且有催化？ |
| C | GARP 成长型 | 核心 | 增长是否真实、可持续、未被充分定价？ |
| D | 产业链爆发型 | 核心 | 是否处在真实瓶颈，并能转化为利润？ |
| E | 落难白马型 | 核心 | 困境是暂时的，还是护城河永久受损？ |
| F | 垄断资源型 | 核心 | 是资源优势 + 供给约束，还是普通周期 beta？ |
| G | 股东回报型低估 | 准核心 | 现金、分红、回购是否能兑现价值？ |

### 5.1 A: DCF / 现金流复利型

入选标准：

1. 商业模式简单且持久。
2. 自由现金流稳定。
3. 长期 ROE / ROIC 高于行业平均。
4. 资本开支可控。
5. 有提价权或成本转嫁能力。
6. 毛利率、净利率稳定或提升。
7. 管理层资本配置理性。
8. 未来 5-10 年被技术替代的概率低。
9. base case 下预期收益有吸引力，不依赖估值扩张。

卖出或降级：

1. ROIC 结构性下降。
2. 行业需求进入结构性下行。
3. 管理层开始低效扩张。
4. 分红或回购纪律恶化。
5. 估值已透支未来多年增长。
6. 预期收益明显低于其他机会。

### 5.2 B: 烟蒂深度低估型

入选标准：

1. 市值低于保守清算价值或资产价值。
2. 资产大概率真实。
3. 主业不会持续大量烧钱。
4. 负债结构清晰。
5. 没有重大关联交易或治理风险。
6. 有明确催化剂。

保守清算价值：

```text
现金 + 0.8 * 应收账款 + 0.5 * 存货 + 0.3 * 固定资产 - 总负债
```

可接受催化剂：

1. 回购。
2. 特别分红。
3. 控股股东或管理层增持。
4. 资产出售。
5. 业务剥离。
6. 私有化。
7. 行业周期触底。
8. 债务重组。
9. 管理层更换。
10. 政策变化。

仓位限制：

| 情况 | 仓位 |
| --- | ---: |
| 财务透明、净现金、回购或分红明确 | 5%-10% |
| 财务一般、存在一定低估 | 3%-5% |
| 困境严重、主要靠反转 | 1%-3% |
| 报表复杂或治理风险高 | 不买 |

时间止损：18-24 个月没有催化进展，退出。

### 5.3 C: GARP 成长型

入选标准：

1. 收入增长来自真实需求和市占率提升，而不是单纯涨价周期。
2. 市占率提升。
3. 毛利率稳定或提升。
4. 现金流随利润改善。
5. 竞争格局没有快速恶化。
6. 增长不依赖持续融资。
7. 估值没有完全定价完美未来。
8. 至少一个核心财务指标超预期。

更严格的定义：

```text
GARP = 可防守增长 + 合理估值。
只有增长，没有护城河，不是 GARP。
只有估值不贵，但增长容易被竞争复制，也不是 GARP。
```

GARP 公司必须证明其增长不容易被资金复制：

1. 新进入者即使有资金，也难以快速复制品牌心智、渠道深度、客户验证、技术积累、供应链效率或规模成本。
2. 高利润不会轻易引来大量竞争者并迅速压低回报率。
3. 公司增长能够转化为经营现金流，而不是长期依赖营销费用、补贴或融资。
4. 公司相对同业有明确优势，而不是只处于行业整体增长 beta 中。
5. 稳定期 ROIC / ROE 应能高于行业平均，并且不是由短期价格周期或费用压缩造成。

以下公司不能归为 GARP：

1. 低壁垒消费品、普通零售、普通食品、简单制造、OEM/ODM、代工出口或流量型品牌。
2. 只靠投放、铺渠道、抢份额获得增长，但产品和品牌容易被复制。
3. 毛利率或净利率一旦较高，就会迅速吸引新玩家进入。
4. PE 约 20-25 倍，但护城河弱、增长可复制、现金流质量一般。
5. 与更强同业相比没有差异化优势，只是跟随行业增长。

PEG 只能作为粗筛。增长率必须自己拆：

```text
行业空间 * 市占率 * 价格趋势 * 利润率趋势
```

卖出信号：

1. 收入增速明显下台阶。
2. 毛利率连续恶化。
3. 销售费用率上升但收入不增长。
4. 应收账款或存货异常增长。
5. 价格战出现。
6. 市场空间被证伪。
7. 估值已经要求持续完美执行。

### 5.4 D: 产业链爆发型

这是主要进攻策略。

必须同时具备三张票：

| 票据 | 要求 |
| --- | --- |
| 需求票 | 下游需求真实且有预算 |
| 瓶颈票 | 公司所在环节供给稀缺、扩产困难或认证严格 |
| 利润票 | 瓶颈能明显转化为上市公司收入和利润 |

证据等级：

| 等级 | 证据 | 动作 |
| ---: | --- | --- |
| 0 | 论坛、传闻、概念 | 仅记录 |
| 1 | 行业趋势明确，公司受益未验证 | 观察池 |
| 2 | 公告、送样、合作、产能建设 | 可建战术仓 |
| 3 | 客户验证、订单、收入开始体现 | 可进准核心 |
| 4 | 财报确认收入、毛利率、利润 | 可升核心 |
| 5 | 持续超预期，护城河强化 | 可核心持有 |

可以追高的条件：

1. 下游需求仍在上修。
2. 订单、客户或产品价格继续改善。
3. 未来利润预期上修速度快于股价上涨速度。
4. 当前估值相对 2-3 年正常化利润并不极端。
5. 供需仍紧。
6. 竞争对手没有快速追上。

不能追高的情况：

1. 只有题材，没有订单。
2. 只有送样，没有收入路径。
3. 股价涨幅远大于盈利上修。
4. 同行业无差别乱涨。
5. 管理层高位融资或减持。
6. 估值已计入极端 bull case。

### 5.5 E: 落难白马型

先判断下跌类型：

| 类型 | 含义 | 动作 |
| --- | --- | --- |
| 杀估值 | 公司没坏，只是以前太贵 | 观察，等右侧 |
| 杀业绩 | 公司短期受损，但商业模式未坏 | 等业绩企稳 |
| 杀逻辑 | 护城河或商业模式永久受损 | 放弃 |

买入标准：

1. 估值处于历史低位。
2. 现金流仍健康。
3. 核心业务没有结构性破坏。
4. 利润预期停止下修。
5. 管理层改善资本配置。
6. 股价不再创新低。
7. 出现右侧放量和趋势确认。
8. 政策或行业环境边际改善。

不能因为好公司跌很多就买。

### 5.6 F: 垄断资源型

入选标准：

1. 储量大且质量高。
2. 成本处于行业低位。
3. 矿山寿命长。
4. 有产能增长或储量替代能力。
5. 供给受资源、政策、环保、资本纪律约束。
6. 资产负债表可控。
7. 管理层并购和开发能力强。
8. 分红或回购有意义。

周期测试：

```text
上一轮行业低谷中，公司是否仍能盈利并产生现金流？
```

卖出或降级：

1. 商品价格进入极端乐观区间。
2. 行业大规模扩产。
3. 公司在周期顶部高价并购。
4. 政策限制利润。
5. 分红纪律恶化。
6. 股价已经完全反映 bull case。

### 5.7 G: 股东回报型低估

入选标准：

1. 净现金有意义。
2. 主业不大量烧钱。
3. 市值低于现金加保守主业价值。
4. 负债低。
5. 经营现金流为正。
6. 回购、分红或特别分红有持续性或较大概率。
7. 治理结构足够清晰。
8. 没有持续融资摊薄。
9. 没有明显关联交易风险。

仓位参考：

| 情况 | 仓位 |
| --- | ---: |
| 现金真实、回购分红明确、主业稳定 | 5%-10% |
| 主业弱但不烧钱 | 3%-5% |
| 现金多但治理弱 | 0%-3% |
| 现金真实性存疑 | 不买 |


## 7. 评分系统

评分用于排序研究优先级，不能自动触发买入。

| 维度 | 权重 |
| --- | ---: |
| 商业模式 / 护城河 | 15 |
| 行业空间 / 周期位置 | 15 |
| 竞争格局 / 行业地位 | 15 |
| 财务质量 | 10 |
| 管理层 / 资本配置 | 10 |
| 估值安全边际 | 15 |
| 催化剂 / 右侧信号 | 10 |
| 风险透明度 | 10 |
| 合计 | 100 |

| 分数 | 动作 |
| ---: | --- |
| 85-100 | 核心候选 |
| 75-84 | 准核心候选 |
| 65-74 | 战术候选 |
| 50-64 | 零仓位观察 |
| <50 | 删除或暂停 |

高分不等于重仓。仓位还要看估值、市场状态、组合集中度和杠杆。

## 8. 估值与情景分析

核心和准核心必须有 bear/base/bull case。

| 情景 | 必须回答 |
| --- | --- |
| Bear | 需求、订单、利润率、估值或政策不及预期时，可能亏多少 |
| Base | 正常兑现下，合理利润和估值是多少 |
| Bull | 需求、利润率和估值超预期时，可能到什么位置 |

最低赔率要求：

```text
base case 上涨空间 / bear case 下跌空间 > 2
```

高波动产业链股最好大于 3。稳定现金流公司可以接受 1.5-2。

反向预期检验：

核心与准核心在写三情景之前，先从当前市值反推市场隐含的增速或利润假设，明确"市场已定价什么、预期差在哪"。如果自己的 base case 与市场隐含假设几乎相同，说明没有预期差，买入理由不成立；如果差距来自自己更乐观的假设，必须写明证据。

## 9. 买入规则

### 9.1 通用买入清单

只有全部满足时才能买：

1. 已进入观察池。
2. 已完成研究卡片。
3. thesis 清晰。
4. 已写 bear/base/bull case，赔率达标。
5. 有右侧信号，或极度低估且有催化剂。
6. 已写明什么情况证明自己错了。
7. 已设定目标仓位。
8. 已设定加仓、减仓、清仓条件。
9. 已完成催化剂地图，每条催化剂可追踪、可验证、有预计时点。
10. 已对前 2-3 个高概率高影响风险写好预设应对。
11. 已核对买入后组合暴露：单票、产业链、大行业上限，以及与现有持仓共享的隐藏 beta。

### 9.2 底部反转买入

适用于落难白马、低估反转、周期底部。

以下条件满足至少 4 条才可以买：

1. 3-6 周不创新低。
2. 股价形成更高低点。
3. 放量站上 60 日线。
4. 回踩不破突破平台。
5. 板块同步改善。
6. 最新财报没有继续恶化。
7. 出现回购、订单、政策、价格、利润率等催化剂。
8. 股价强于指数和行业。

三批建仓：

| 批次 | 触发条件 | 仓位 |
| --- | --- | ---: |
| 第一批 | 底部反转条件成立 | 目标仓位 1/3 |
| 第二批 | 回踩平台不破且缩量 | 目标仓位 1/3 |
| 第三批 | 创新高或出现第二个基本面验证 | 目标仓位 1/3 |

如果股价连续两个交易日跌回平台，或跌破前低，退出战术部分。

### 9.3 突破 / 产业链买入

适用于产业链爆发和强成长。

触发条件：

1. 成交量至少为 20 日均量 2 倍。
2. 突破至少 3 个月平台，或创 52 周新高。
3. 板块和龙头同向确认。
4. 24 小时研究确认需求、瓶颈和利润转化。

仓位路径：

| 阶段 | 证据 | 仓位 |
| --- | --- | ---: |
| 初步异动 | 板块启动，公司强于同业 | 1%-3% |
| 初步验证 | 公告、送样、合作、产能证据 | 3%-5% |
| 强验证 | 客户、订单、收入开始体现 | 5%-10% |
| 财报兑现 | 利润、毛利率、订单持续超预期 | 10%-20% |
| 长期确认 | 瓶颈持续，护城河强化 | 核心仓位 |

不要等完美回调，也不要在验证前一次买满。

### 9.4 24 小时异动响应

观察池股票突然放量上涨后，24 小时内回答：

1. 为什么涨？
2. 是个股事件，还是行业事件？
3. 事件真实吗？
4. 是否有公告、财报、客户侧证据？
5. 公司是否在瓶颈环节？
6. 该业务能贡献多少利润？
7. 市场此前是否低估了利润弹性？
8. 产业链中谁最受益？
9. 龙头是谁？
10. 当前估值相对未来利润是否仍可接受？

结论只能三选一：

1. 买入战术仓。
2. 参数化等待。
3. 放弃。

参数化等待必须写清：

```text
回调到 X 价格买 Y 仓位，N 个交易日内有效。
如果 N 日内不回调反而创新高，按突破规则买第一批。
```

真实触发后，不允许用“继续观察”作为结论。

### 9.5 观察池事件动量

这是用于捕捉“高质量股票突然被市场重新定价”的战术策略，典型形态是：

```text
高质量观察池公司 -> 某日放量大涨 -> 快速确认催化剂 -> 追入战术仓 -> 延续则加仓 -> 动量衰竭或逻辑证伪则卖出。
```

该策略只允许用于已经进入观察池、重点观察池或准买入池的公司。陌生股票突然大涨，不适用本策略。

适用场景：

1. 落难白马出现底部放量修复。
2. GARP 公司业绩、订单、利润率或行业景气突然超预期。
3. 产业链瓶颈公司出现需求、订单、价格、客户认证或政策催化。
4. 股东回报型低估公司出现回购、增持、特别分红、资产重估或价值兑现催化。

买入触发必须同时满足：

1. 公司此前已经通过观察池质量筛选。
2. 单日成交量至少为 20 日均量的 2 倍。
3. 股价突破关键平台、重要均线或阶段新高。
4. 能在 24 小时内找到真实催化剂，而不是只有市场传闻。
5. 当日或次日仍有足够成交承接，不是一字板、脉冲板或无量拉升。

仓位路径：

| 阶段 | 触发 | 仓位 |
| --- | --- | ---: |
| 第一笔 | 放量突破且催化剂初步真实 | 目标仓位 1/3，通常 1%-3% |
| 第二笔 | 继续放量上涨，或横盘 2-5 个交易日不跌回启动平台 | 目标仓位 1/3 |
| 第三笔 | 第二次突破、研报/公告/财报进一步验证，或板块龙头同步加强 | 目标仓位 1/3 |

事件动量仓属于战术持仓：目标仓位默认不超过战术持仓单只上限（v20起为5%总资产，允许一笔建满，前置买入清单+割肉价+1.5%净资产单笔风险校验）。只有当催化剂进入订单或财报级验证、并完成完整重估后，才允许按突破/产业链买入路径（9.3）升级到更高仓位区间。

持有条件：

1. 股价不跌回启动平台。
2. 回调缩量，上涨放量。
3. 板块或同类高质量公司同步确认。
4. 催化剂没有被证伪。
5. 盈利预期或估值修复空间仍未被完全定价。

卖出规则：

1. 跌破启动日低点，退出。
2. 连续两个交易日跌回突破平台，退出。
3. 放量滞涨、长上影后次日不能修复，减仓。
4. 5 日线和 10 日线同时失守，且没有新增基本面证据，减仓或退出。
5. 催化剂被证伪，立即退出。
6. 阶段涨幅已经反映 bull case，且后续没有盈利上修，分批止盈。

禁止事项：

1. 禁止把失败的事件动量仓转成长期仓。
2. 禁止因为“公司质量不错”而忽视短线失败信号。
3. 禁止在没有真实催化剂时只凭大涨追入。
4. 禁止在第一笔买入后无验证地补仓摊平。
5. 禁止把启动价当作绝对底部；启动价跌破时必须重新评估。

该策略的目标是吃到预期差被市场连续定价的主升早段，而不是预测完整顶部。买入靠异动和催化剂，持有靠延续，卖出靠动量衰竭和 thesis 证伪。

## 10. 加仓规则

只有 thesis 变强时才能加仓，不能因为股价下跌而加仓。

允许加仓：

1. 财报超预期。
2. 订单或客户验证改善。
3. 毛利率改善。
4. 行业景气度上修。
5. 突破后回踩不破。
6. 竞争格局改善。
7. 估值仍有空间。

限制：

1. 没有新增证据，不摊平亏损。
2. 每次加仓原则上不大于上一笔，除非重新完整评估。
3. 加仓后不得违反组合集中度限制。

## 11. 减仓与卖出规则

减仓和卖出依据 thesis、估值、风险、机会成本、催化剂状态和右侧失败，不依据浮盈浮亏。

### 11.1 Thesis 证伪卖出

以下情况卖出或降级：

1. 订单没有兑现。
2. 客户验证失败。
3. 利润率没有改善。
4. 行业需求被证伪。
5. 竞争对手进入速度超预期。
6. 技术路线变化。
7. 政策逻辑变化。
8. 管理层表述与财务数据矛盾。

### 11.2 估值过高卖出

以下情况减仓：

1. 当前价格已经反映 bull case。
2. 未来 2-3 年利润已被充分定价。
3. 股价上涨远快于盈利上修。
4. 板块进入普遍泡沫。
5. 利好不断出现但股价不再上涨。

### 11.3 机会成本卖出

只有新机会评分至少比旧持仓高 15 分，或旧持仓 thesis 明显变弱，才允许大幅切换。

目的是避免频繁追逐短期热点。

### 11.4 风险控制卖出

以下情况减仓：

1. 单只股票超过 30%。
2. 同一产业链超过 40%。
3. 同一大行业超过 60%。
4. 杠杆超过安全区间。
5. 多个持仓共享同一个隐藏 beta。

### 11.5 催化剂兑现卖出

催化剂兑现后必须重新评估。如果没有下一轮催化或盈利上修，减仓。

催化剂包括：

1. 政策落地。
2. 订单公告。
3. 产品发布。
4. 财报兑现。
5. 指数纳入。
6. 并购重组完成。
7. 分红或回购执行。

### 11.6 技术破位卖出

适用于战术持仓和产业链突破：

1. 跌破突破平台。
2. 跌破 60 日线后无法收回。
3. 放量下跌。
4. 反弹无量。
5. 板块龙头开始破位。
6. 公司没有新的基本面证据。

### 11.7 治理 / 财务红旗卖出

严重时卖出或排除：

1. 控股股东资金占用。
2. 复杂关联交易。
3. 应收账款异常。
4. 存货异常。
5. 经营现金流长期低于利润。
6. 审计机构异常更换或非标意见。
7. CFO 频繁更换。
8. 大股东高质押。
9. 高位融资。
10. 重要股东持续减持。
11. 管理层解释与数据矛盾。

## 12. 盈利后减仓与十倍股保护

### 12.1 原则

“盈利后逐步慢慢减仓，最终不清仓”的想法是合理的，但只适用于一种情况：

```text
thesis 仍然成立，并且持续被验证。
```

如果 thesis 被证伪、财务质量恶化、治理风险暴露，底仓规则失效，可以并且应该清仓。

### 12.2 开始减仓的条件

不能只因为盈利就减仓。至少满足以下一条才启动部分止盈：

1. 仓位超过计划上限。
2. 估值已经反映 bull case。
3. 股价上涨远快于盈利上修。
4. 产业链供给即将明显放松。
5. 商品或板块情绪进入极端乐观。
6. 趋势保护信号触发。
7. 出现评分高 15 分以上的新机会。

### 12.3 慢减仓表

| 相对持仓成本盈利 | thesis 仍成立时的动作 |
| ---: | --- |
| +50% | 不自动卖出，只复查仓位和 thesis |
| +100% | 若估值或仓位偏高，卖出剩余股数的 10%-15% |
| +200% | 若盈利上修没有跟上，再卖出剩余股数的 10%-15% |
| +300%-500% | 每新增约 100% 涨幅，最多卖出剩余股数的 10%-15% |
| >+500% | 停止机械卖出，改用趋势和 thesis 保护 |
| 10 倍 | thesis 未破时保留永久底仓 |

约束：

1. 两次止盈至少间隔 20 个交易日。
2. 每次卖的是剩余股数的一部分，不是原始仓位的一部分。
3. 单月止盈不得超过剩余股数的 25%，除非违反风险限制。
4. thesis 仍成立时，不卖掉最后底仓。

### 12.4 底仓比例

| 股票类型 | 保留底仓 |
| --- | ---: |
| 现金流复利型 | 账户 5%-10%，或原始股数 20%-30% |
| 产业链爆发型 | 账户 3%-8%，或原始股数 15%-25% |
| 垄断资源型 | 供给约束仍在时，账户 5%-10% |
| 落难白马修复后 | 预期收益仍可接受时，账户 3%-8% |
| 烟蒂 / 事件驱动 | 不强制保留，价值兑现后退出 |

烟蒂和事件驱动是例外。价值兑现后应该退出，除非重新按另一个策略完整评估。

### 12.5 大盈利后的趋势保护

| 策略 | 保护线 |
| --- | --- |
| 现金流复利型 | 月收盘跌破 12 月均线，核心降为准核心 |
| 产业链爆发型 | 周收盘跌破 30 周线，先减半 |
| 战术持仓 | 跌破突破平台，退出 |
| 资源股 | 商品价格极端乐观且供给扩张，减仓 |

这条规则的目标是避免过早卖飞，同时避免把巨大浮盈完整还给市场。

## 13. 组合级风险规则

### 13.1 回撤预算

| 账户从高点回撤 | 动作 |
| ---: | --- |
| -8% | 移除杠杆 |
| -12% | 黄色预警：禁止升级、禁止新增杠杆，只能在强证据下加仓 |
| -20% | 红色预警：总仓位降至 <=50%，暂停新买入 10 个交易日，完成复盘 |

### 13.2 单笔风险

战术和右侧新开仓必须满足：

```text
仓位比例 * 止损距离 <= 账户净值的 1.5%
```

核心仓不设简单价格止损，但浮亏 20% 必须重写 thesis，浮亏 30% 必须有新增证据，否则降级。

## 14. 工作流

### 14.1 每日

只跟踪：

1. 当前持仓。
2. 准买入池。
3. 重点观察池前 10 名。
4. 重大行业事件。
5. 官方公告和财报。
6. 关键产业价格、订单、政策和产能数据。

### 14.2 每周

回答：

1. 核心持仓 thesis 是否变化？
2. 准核心是否可以升级？
3. 战术仓是否应该删除？
4. 观察池是否出现买入触发？
5. 哪些公司评分上升？
6. 哪些公司评分下降？
7. 当前市场是强势、弱势还是震荡？
8. 杠杆是否过高？
9. 行业暴露是否集中？
10. 下周最需要跟踪的 5 只股票是什么？

### 14.3 每季度

财报季后更新：

1. 收入。
2. 毛利率。
3. 净利润。
4. 经营现金流。
5. 应收账款。
6. 存货。
7. 资本开支。
8. 管理层指引。
9. 订单和在手订单。
10. 与 thesis 的偏差。

### 14.4 每年

复盘：

1. 收益率相对基准。
2. 最大回撤。
3. 胜率。
4. 盈亏比。
5. 规则违反次数。
6. 错过的大牛股。
7. 最大可避免亏损。
8. 主动投资是否仍创造价值。

如果连续四个季度累计跑输基准超过 10 个百分点，必须降低主动仓位或在书面复盘后修订体系。

## 15. 禁止事项

1. 禁止因为亏损而拒绝减仓。
2. 禁止因为盈利而机械卖出。
3. 禁止用杠杆买题材股。
4. 禁止在没有财报、订单或客户验证时重仓。
5. 禁止没有 bear/base/bull case 就买入。
6. 禁止把起涨点当作安全底。
7. 禁止因为错过而报复性追高。
8. 禁止把失败的战术仓变成长线仓。
9. 禁止没有新增证据就摊平亏损。
10. 禁止在亏损当周修改体系规则。

## 16. 一票否决

以下情况严重时不买，已持有则退出：

1. 财务造假嫌疑。
2. 控股股东资金占用。
3. 复杂关联交易。
4. 应收账款增速远高于收入。
5. 存货大增但收入不增长。
6. 经营现金流长期落后于利润。
7. 审计机构异常更换或非标意见。
8. 重要股东持续减持。
9. 高位融资。
10. 频繁并购导致高商誉。
11. 核心业务持续亏损且无可信改善。
12. 故事多，财报验证少。
13. 只有传闻，没有公告或客户验证。
14. 公司历史上多次蹭热点失败。

## 17. 行为守则：既往病历与重蹈覆辙拦截

本节把真实亏损案例和 2026-07-11 策略路线决议固化为红线。执行代理（大模型）收到命中红线的指令时，必须先纠正劝阻、再按协议留痕，不得直接照办，也不得假装合规。

### 17.1 既往病历

**病历一：迈瑞医疗 2025-09（左侧 + 杠杆 + 摊平的完整失败样本）**

事实经过：2025 年 9 月以约 240 元全仓买入，融资杠杆约 1.75x、担保率约 200%；下跌途中在约 200 元和约 180 元两次以场外资金加仓，无任何新增证据，持有依据是管理层"业绩企稳"的口头承诺；2026 年春年报与一季报相继不及预期，约 154 元清仓，单票 -36%。-36% 时若无场外输血，担保率约 128%，已进入 130% 强平区——退出时点几乎交给了券商。

错误分解（每条对应现行规则）：

| # | 错误 | 对应规则 |
| --- | --- | --- |
| 1 | 全仓单票、无仓位上限 | §2.1 持仓层级 |
| 2 | 杠杆做左侧等待，无压力测试 | §4 杠杆规则 |
| 3 | 无新增证据两次摊平 | §10 限制 1 |
| 4 | 管理层口头承诺当证据 | §9.1 清单、§16.12/13 |
| 5 | 无止损、无证伪条件，纠错只能等财报 | §9.1 第 6 条、割肉价制度 |

结构性教训：左侧模式在财报落地之前没有任何机制提示论点已错，市场连续拒绝确认本身就是信息；杠杆账户没有"长期"，杠杆账户的长期等于每一个短期都活着。

### 17.2 策略路线决议（2026-07-11）

背景：净资产约 282 万、融资负债约 218 万（1.77x）、年化收益底线 20%、年新增储蓄约 20 万；比较"估值底座 + 右侧触发"（现行）与"纯左侧买入等待回归"两条路线，两者均无已验证战绩。

决议：**维持现行右侧体系为唯一操作系统；杠杆 ≥1.3x 期间，禁止无信号、无催化剂的左侧买入。**

依据：

1. 生存数学：担保率 229.66% 距 130% 强平区约 -43% 组合跌幅，届时净资产约 -77%；左侧定义上买入下跌中的股票并延长浮亏期，与 1.77x 杠杆互斥。融资利息约为净资产 3.5-4%/年，是左侧等待的负 carry。§13.1 的 -8% 去杠杆线在 1.77x 下对应组合仅约 -4.5%，正常左侧建仓期即会击穿：左侧路线与本体系风控规则数学互斥。
2. 收益钳形：无杠杆左侧优质价值的全周期底数约 10-15% 年化，低于 20% 底线；带杠杆则回到第 1 条。该路线在自身收益要求下自我否决。
3. 行为证据：8/8 持仓体系外建仓、业绩预告后 +31% 追入、自设锁定期防过度交易，均为高行动倾向证据。左侧没有行动配给机制，行动能量会以"价值加仓"名义泄洪（迈瑞 200/180 两笔即是）；右侧把行动配给到信号日。小资金唯一的结构性优势（进出速度）只有右侧能变现。
4. 语录还原：巴菲特"小资金可做 50% 年化"指微型股与特殊情况套利，且其终生反对 margin（"聪明人破产三途：酒、女人和杠杆"）；该语录不构成杠杆左侧的依据。

左侧重启条件（全部满足后，可再议一个无杠杆左侧底仓 sleeve，在此之前本决议持续有效）：

1. 总杠杆 ≤1.1x。
2. 连续两个季度体系内执行（无 off_system 买入）。
3. 接受该 sleeve 预期年化 12-15%。
4. 以股息型低估优质为主，单票 ≤10%、不摊平、保留年报级 thesis 复核。

注意：买入低估好公司的合法通道现已存在——§9.1 第 5 条"极度低估且有催化剂"+ 工作流 §13 纯左侧首笔限 5%。本决议否决的不是价值投资，是"杠杆下不等确认就买、买了不设止损"。

### 17.3 红线清单与拦截协议

红线清单（命中任一即触发拦截）：

| # | 红线 | 病历对应 |
| --- | --- | --- |
| R1 | 杠杆 ≥1.3x 时，既无右侧信号也无已核实催化剂的左侧买入（理由仅为"好公司+便宜/跌多了"） | 迈瑞 2025-09 入场 |
| R2 | 对浮亏持仓在无新增证据下加仓（摊平） | 迈瑞 200/180 两笔 |
| R3 | 以管理层口头表态（董秘承诺、调研纪要式"企稳/向好"）作为买入或持有的主要证据 | 迈瑞持有理由 |
| R4 | 下调或取消割肉价；以"长期投资/价值投资"为由移除止损或趋势保护 | 迈瑞无止损结构 |
| R5 | 单票超上限、全仓集中，或在现有杠杆上继续加杠杆 | 迈瑞全仓 |
| R6 | 把触发退出信号的战术/事件仓改叙事为"长期价值仓" | §9.5 禁止事项 1、§15.8 |
| R7 | 盈利期/净值新高期提出拆除体系防护层（移除信号层、删除回撤预算、废除割肉制度、整体转左侧） | 2026-07-11 咨询即属此类 |

拦截协议（执行代理逐步执行）：

1. 识别命中红线的指令后不直接执行：先引用红线编号与病历事实（含具体数字）纠正劝阻。
2. 给出体系内替代路径：左侧想法 → §9.1 第 5 条催化剂通道（首笔 5%）；摊平 → 列出所需新增证据清单；R7 → 冷静期后按 §17.2 重启条件评估。
3. 用户接受劝阻 → 决策日志记 `discipline_intervention`（结论=接受劝阻）。
4. 用户坚持 → 用户资金用户有最终决定权，但必须：先按工作流 §10.4 写 `off_system` 前置记录，决策日志记 `discipline_intervention`（结论=用户坚持执行），持仓 notes 如实记体系偏离。执行代理不得假装合规，也不得在完成留痕后拒绝记录执行结果。
5. 方向豁免：降风险方向的指令（减仓、清仓、上调割肉价、偿还融资、降杠杆）永不拦截。
6. 防护层修改冷静期：对体系防护层的结构性修改（R7 类），须在提出之日起至少 5 个交易日后，以书面复盘（引用本节）再次确认，方可执行；与 §15.10 对称——亏损周禁改规则，盈利周的拆除同样需要冷静期。
