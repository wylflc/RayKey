TL;DR · 框架核心

本框架把“是否买入一只股票”拆成 五步流水线：① 业务拆解 + 行业周期 → ② 护城河与竞争格局 → ③ 财务体检 + 收入质量 → ④ 估值定价 + 隐含预期 → ⑤ 交易计划与风控。每一步都设有 "通过 / 警示 / 否决"三色决策门，前一步否决就不必走下一步。最终产出一张可填写的 决策卡：含是否买入、目标价、止损价、仓位、分仓路径和失效条件——拿到就能执行。

适用前提：本框架是"以合理价格买入优质公司并控制下行风险"的价值-成长混合派打法，适合中线（1-12个月）持有。纯短线情绪博弈、纯量化套利不在覆盖范围。对于Biotech/早期亏损/周期拐点等特殊类型公司，第四步设有估值方法适用性检验子模块。

0总览：五步流程图与决策门

五步流程的核心思想：先用前三步筛掉"不该碰的公司"，再用第四步筛掉"该碰但现在太贵的公司"，最后在第五步把符合条件的公司变成可执行的单子。绝大多数亏损不是买错了时机，而是把前三步该淘汰的公司拿到了第五步。

1第一步 · 业务拆解 + 行业周期定位（定性）

目标：30 分钟内搞清楚"这家公司靠什么赚钱、哪块业务决定估值、行业处于什么周期、近期有什么催化"。看不懂就直接放弃——这是第一条防线。

1.1 必答六问

公司画像

  • 主营卖什么产品/服务？一句话能说清吗？
  • 收入结构：哪块业务贡献最大营收？哪块贡献最大利润？
  • 主要客户是谁（To C / To B / To G）？集中度如何？
  • 在产业链什么位置（上游原料/中游制造/下游品牌/渠道）？

赚钱逻辑

  • 商业模式：高毛利低周转？还是低毛利高周转？
  • 增长来源：靠量（份额/新品）还是靠价（提价/升级）？
  • 资本密集度：轻资产还是重资产？扩张要不要持续烧钱？
  • 近 1-2 年最重要的经营变化是什么（新品/并购/转型/管理层变更）？

1.2 业务结构拆解表（套用模板）

业务板块营收占比毛利率增速收入质量是否决定估值趋势判断
主业 A：__________%____%____%经常性/一次性是______
增长业务 B：__________%____%____%经常性/一次性潜在______
边缘/衰退 C：__________%____%____%经常性/一次性否______

提示：决定估值的那块业务（通常是高增长或高毛利板块）才是分析重点，其余板块只需了解趋势即可。“收入质量”列是v2.0新增列：区分经常性收入与一次性收入（如授权首付款、资产处置收益、政府补贴等），避免被一次性收入粉饰的利润误导估值。

1.3 行业周期定位v2.0 新增v2.1 产能与技术周期

同一家公司在行业周期的不同位置，财务数据的含义完全不同。周期股在顶点时财务最好看（高增长低PE），但那是卖出信号。

周期阶段特征财务信号估值特征策略含义
导入期产品刚上市，未来不确定亏损/微利，增速快PS/EV为主，PE失效看渠道和用户增长，不看利润
成长期份额快速扩张，格局未定高增速，利润率上升PEG为主，高PE可接受看增速可持续性和壁垒强度
成熟期份额稳定，竞争格局已定增速放缓，现金流好PE/PB/股息率为主看分红/回购和余量增长点
衰退期业务走下坡，转型不确定收入下滑，利润变薄看PB/清算价值价值陷阱高发，底部不等于便宜
产能/技术周期
v2.1		大规模扩产期末尾 / 去库存尾声 + 新技术标准推动换装潮CapEx增速拐点 / 存货周转率拐点 / 新一代技术规范落地节奏扩产期末尾：ROE/ROIC依然好看但即将价格战；去库存尾声：利润承压但左侧买点临近扩产期末尾→警惕价格战风险，下调Bear目标价；去库存尾声+新技术落地→最佳左侧买点
典型应用：阳光电源（300274）在2021年属于“成长期”，市场给高PE合理；2025年光伏产业已从“成熟期”向“衰退期”过渡，落后产能被淘汰。三生国健（688336）的创新药管线处于“导入期→成长期”跳变，估值方法需随之切换。
v2.1 产能与技术周期研判要点：① 关注行业CapEx增速拐点——当全行业CapEx增速连续2年>20%时，即使当前ROE/ROIC依然好看，也可能即将迎来产能过剩和价格战；② 关注新一代技术标准落地节奏（如5G→5.5G→6G、TOPCon→HJT→钙钛矿、DDR4→DDR5→HBM）——去库存尾声+新技术规范推动换机/换装潮是最佳左侧买点；③ 对比重资产与硬科技行业时，"产能周期"比"生命周期"更精准——同一轮技术迭代中，先动者吃超额利润，后动者吃价格战。

2第二步 · 护城河与竞争格局评估

目标：判断公司有没有"真壁垒"——能持续赚取超额回报（ROIC > WACC）的护城河。无壁垒的公司只能做短线交易，不能做中线投资。

  • 核心高管是否稳定（近 3 年有无重大离职）

2.5 护城河侵蚀风险v2.0 新增

护城河不是静态的。问自己：3年内，什么因素可能摧毁这条护城河？

侵蚀因素示例发生概率影响程度

政策变化集采/反垄断/行业规范调整高/中/低致命/严重/轻微

3.2 质量增长匹配检验（最容易被忽略的红旗）

增长是真好还是靠"堆资产 + 借钱"硬撑出来的？用两个比值一查便知：

增长质量 = 净利润增速 ÷ 净资产增速
  • >1：利润增长快于资本投入 → 内生高质量增长 ✓
  • ≈1：投入和利润同步 → 正常但需关注回报率是否下降
  • <1：投入很多但利润增长慢 → 低效扩张，价值毁灭信号 ✗
ROE 拆解（杜邦）：ROE = 净利率 × 资产周转率 × 权益乘数
  ROE 上升但靠"权益乘数（加杠杆）"撑 → 警惕
  ROE 上升且靠"净利率 / 周转率"提升 → 高质量

3.3 收入质量拆解v2.0 新增

这是v2.0最重要的新增模块。很多公司的利润爆发并不可持续——一次性收入（BD授权、资产处置、政府补贴）会让PE看起来极便宜，但前瞻性利润可能是阶跃式上升后断崖式下降。

收入类型特征财务影响市场定价含义
经常性收入产品销售、稳定服务费可持续、可外推应用正常PE估值
一次性收入授权首付款、资产处置、补贴不可持续、无法外推应从PE中剔除后评估
里程碑款BD交易的后续款项有不确定性、但可能很大作为期权价值单独估算
检验方法：剔除一次性收入后的“经常性利润”
经常性净利润 = 报告期净利润 − 一次性收入税后贡献
经常性PE = 当前市值 ÷ 经常性净利润

典型案例：三生国健（688336）2025年营收42亿（+252%），利润29亿（+317%），但其中约28亿是辉瑟授权首付款。剔除后经常性利润仅约2-3亿，TTM PE 18x→前瞻PE 80x。如果不做收入质量拆解，就会得出“PEG 0.06极便宜”的荒谬结论。

决策门 ③ · 财务有红旗吗？收入质量如何？ 通过 四维全绿 + 增长质量 >1 + 经常性利润稳定 + 资本配置合理 → 进入第四步
警示 1-2 项黄牌（如现金流偶尔 <1）/ 一次性收入占比高 → 谨慎继续，估值上要求更大折扣
否决 出现红旗（经营现金流长期为负 / ROIC<资金成本 / 财务疑点 / 经常性利润持续下滑） → 放弃

4第四步 · 估值与定价 + 隐含预期拆解

目标：回答四件事——该用什么估值方法、当前估值反映了市场什么预期、当前价格相对合理价值是贵还是便宜、这个估值方法对这家公司是否有盲区。取数务必拉 5-10 年历史算分位，3 年看不出估值中枢。

4.1 公司类型 → 估值方法选择

公司类型特征主用方法辅助验证
高成长增速>25%、赛道新PEG / PS / EV-EBITDAPE（看增速匹配）
稳健成长增速 10-25%、护城河稳PE / PEGDCF
周期盈利随周期大起大落PB / 周期中枢 PE盈利拐点判断

Biotech/早期亏损管线驱动价值、无利润或利润波动大rNPV（风险调整净现值）/ 管线期权估值PS / 同业BD交易对标

4.2 估值位置三重判断

① 历史纵向分位

<30% 偏低
30-70% 中性
>70% 偏高

② 行业横向对比

与同业可比公司比 PE/PB

低于行业中位 + 质地更好 → 折价机会

③ 预期拆解

当前股价隐含了什么预期

低估值隐含周期下行/治理折价
说清"已定价什么"是关键

4.3 三情景估值法 → 目标价区间

不要给单一目标价，给 Bear / Base / Bull 三情景，分别对应悲观 / 中性 / 乐观假设，最后加权（如 25% / 50% / 25%）得到合理价值中枢。

情景假设（增速/利率/份额）估值方法对应目标价较现价空间
Bear 悲观增速下修 / 景气下行__________ 元____%
Base 中性维持当前趋势__________ 元+____%
Bull 乐观超预期 / 逻辑重估__________ 元+____%
加权合理价值25% / 50% / 25%____ 元+____%
DCF 参数边界（若用 DCF）：WACC 通常 8%-12%，永续增长率 2%-3%（不超过 GDP 增速）；terminal value 占比 >70% 要警示（说明估值过度依赖远期假设）。务必做敏感性分析。

4.4 反向DCF / 隐含预期拆解v2.0 新增

正向估值是“我认为值多少”，反向DCF是“市场认为值多少”。二者对比才能发现预期差。

反向DCF逻辑
当前市值 = 未来现金流的折现值
→ 反推市场隐含的永续增长率和近期增速
→ 对比自己的判断：市场隐含的增速是否合理？

如果市场隐含增速 = 5%，你认为可达 15% → 预期差在你（低估）
如果市场隐含增速 = 20%，你认为只能 10% → 预期差在市场（高估）

实操方法：用当前市值反推DCF模型，得到隐含的近期增速和永续增长率，再判断这个增速是否可实现。也可以用更简单的PEG反推：当前PE ÷ 合理增速 = PEG，PEG<1说明市场低估了增速。

4.5 估值方法适用性检验v2.0 新增

每种估值方法都有盲区。在用某种方法得出“便宜”或“贵”的结论前，先检验该方法对这家公司是否有效。

估值方法适用场景失效场景替代方法
PE / PEG稳定盈利的成长型公司周期股、亏损公司、利润含大量一次性收入PB / 周期中枢PE / rNPV
PB重资产、金融、周期股轻资产公司（PB失去意义）PE / PS
PS亏损但增长快的公司利润率极低且无改善趋势EV-EBITDA / PEG
DCF现金流可预测的成熟公司现金流为负或极不稳定相对估值法
rNPVBiotech管线估值已有成熟产品的公司PE / PS
核心原则：如果估值方法选错，结论一定错。典型错误：用PE估值周期股（周期顶点PE低但其实是卖点）、用PE估值Biotech（利润不能反映管线价值）、用PE估值含一次性收入的公司（如三生国健）。
决策门 ④ · 当前价格有性价比吗？估值方法适用吗？ 通过 加权合理价值较现价有 >20% 上行空间 + 历史分位 <50% + 估值方法适用 + 反向DCF隐含增速可实现 → 进入第五步（可买入）
警示 上行空间 10-20% / 分位中性 / 估值方法有局限 → 观望，等回调或基本面强化
否决 上行空间 <10% / 分位 >70% / 已透支预期 / 估值方法不适用 → 不买，等更低价或放弃

5第五步 · 交易计划与风控（含技术形态）

目标：把"值得买的结论"变成"可执行的单子"——明确什么价位买、买多少、什么价位卖、什么条件止损。没有止损位的计划不是计划。

5.1 先判断当前交易位置 + 形态分析原改进方法升级v2.1 资金流向

前四步解决了"买什么"和"值多少"，这一步解决"何时扣动扳机"。形态分析不是独立步骤，而是内嵌在交易位置判断中，使买点更具象化。形态只有配合量能才有意义——量在价先。

位置基本特征典型形态量能信号微观筹码/资金流向 v2.1策略倾向
低位启动
（试错仓介入）		长期下跌后横盘筑底形态：W底、头肩底、圆弧底放量突破颈线 + 30日低点ETF净流入 / 龙虎榜机构买盘试错仓介入；右侧放量突破颈线是最佳买点

v2.1 资金流向确认原则：① 形态突破必须有增量资金配合——单纯的形态突破如果没有量能放大和资金流入配合，常常演变成假突破；② A股核心资产/科技主线：观察ETF净流入、龙虎榜机构买盘作为右侧确认信号；③ 港股通标的：观察南向资金连续净流入且持股比例上升作为突破确认；④ 资金流向与形态共振时方可重仓，资金流向与形态背离时降低仓位或观望。
形态分析使用原则：① 形态只是辅助工具，不能取代基本面——基本面不过关的公司，形态再好也不买；② 多级别共振：日线和周线形态共振时信号最强；③ 设定形态失败的应对方案：假突破（突破后缩量跌回）应在3日内认错止损，不扛单。

5.2 买入：分仓路径（不要一把梭）

阶段仓位触发条件
试错仓10-20%首次信号出现（逻辑催化 + 技术确认 + 形态验证）
确认仓+20-30%走势符合预期（放量上行 / 站稳关键位）
加仓仓+20-30%强势确认（突破前高 / 业绩兑现）
上限≤70%极强势 + 环境支持（非满仓，留余地）

5.3 止损：三维止损框架 + 形态破位（核心风控）原改进方法升级

① 价格止损 + 形态破位

跌破关键支撑位或幅度止损

• 跌破前低 / 20日线 / 关键位
• 核心形态破坏：突破箱体后缩量跌回（假突破）、跌破W底颈线
• 形态破坏时，哪亏不到5%也应执行纪律
• 幅度止损 -5% 至 -8%（按波动率调整）
确认方式要写清：收盘破位 / 盘中破位 / 连续破位

② 逻辑止损

核心催化失效则走人

• 利好落空 / 公告澄清
• 板块龙头跌停 / 板块跌 5%+
• 放量破位（量能放大 1.5 倍 + 破关键位）
• 业绩证伪买入逻辑

③ 时间止损

横盘占用资金太久则退

• 短线 5-10 个交易日未验证
• 中线 1-2 个月未兑现逻辑
• 弱势震荡超阈值 → 降仓或退出
• 催化剂地图中的催化剂超期未兑现

止损纪律红线：触发后禁止重新评估持有理由——评估应在触发前完成。最常见的亏损放大就是"触发止损后又找理由扛着"。同时区分噪音 vs 真失效：盘中插针快速收回通常是诱空，连续多日趋势性破位才是真失效。形态破坏是最客观的止损信号——它不依赖你的成本价或心态。

5.4 止盈：分批落袋 + 技术信号原改进方法升级

  • 到达 Base 目标价：卖出 1/3 回收成本，留底仓吃主升
  • 到达 Bull 目标价：再卖 1/3，剩 1/3 做潜在第二段（设移动止损保护）
  • 情绪止盈：换手 >15% + 放量滞涨 → 提前减仓
  • 形态止盈（新增）：出现顶背离（如MACD顶背离）或跌破上升趋势线 → 右侧减仓信号
  • 形态破坏止盈（新增）：跌破M头颈线 / 头肩顶颈线 + 换手率放大 → 果断落袋
  • 若强势穿越 Bull 目标价：上调目标 + 移动止损跟随，不轻易下车

5.5 仓位计算公式 + 信心加权v2.0 信心加权新增

仓位由可承受风险反推，不由主观信心正推

单笔风险预算 = 账户总资金 × 单笔最大可承受亏损%（通常 1-2%）
基础仓位 = 单笔风险预算 ÷ (买入价 - 止损价) ÷ 买入价

信心加权系数（v2.0新增）
实际仓位 = 基础仓位 × 信心系数
信心系数由五步决策门通过情况决定：
  • 四步全通过 + 形态共振 = 1.2（高信心，可适当加仓）
  • 三步通过 + 一步警示 = 1.0（标准仓位）
  • 两步通过 + 两步警示 = 0.5（低信心，仅试错仓）
  • 任一步否决 = 0（不买）

例：账户 10 万，单笔最大亏 2%（2000 元），买入价 50 元，止损价 46 元（止损距离 8%）
→ 基础仓位 = 2000 ÷ 8% = 25,000 元（即账户 25%）
→ 若四步全通过 + 形态共振，实际仓位 = 25,000 × 1.2 = 30,000 元（30%）

止损距离越远，仓位越小——而不是放宽纪律。没有止损位就无法计算仓位。信心加权不是说“感觉好就多买”，而是说“决策门通过越多，分析越充分，容错率越低，才能适当加仓”。

5.6 失效条件（高质量计划必备）

写下"什么情况出现就承认这笔交易错了"——避免死扛：

  • 催化证伪：核心利好未兑现 / 被公告澄清
  • 板块转弱：龙头负反馈 / 板块退潮
  • 放量破位：跌破关键位且量能放大
  • 次日反馈差：买入后次日无延续 / 高开低走大阴
  • 环境不支持：大盘转弱 / 风格切换不利于该类资产
  • 形态破坏：核心形态颈线跌破（新增）

⚠风险因子矩阵v2.0 新增v2.1 估值锚失效

!常见陷阱与红线

🚫 分析阶段陷阱

  • 只看 1-2 年数据就被周期误判——至少拉 5 年
  • 把"受益于行业景气"当投资逻辑——必须具体到影响路径
  • 跨公司比 PE 不对齐财年口径（财年 vs 自然年）
  • 高估值直接判"贵"——要看增速与质量是否匹配（PEG）
  • 低估值直接判"便宜"——可能是价值陷阱（周期下行/治理差）
  • 把研报当一手数据——研报需核实，优先采信交易所/年报
  • 不区分经常性与一次性收入——用含一次性收入的PE来估值会严重失真
  • 用错估值方法——用PE估周期股/Biotech/亏损公司
  • 不看行业周期——周期股顶点PE低是卖点不是买点

🚫 交易阶段陷阱

  • 无止损位就买入——没有止损就没有仓位计算
  • 触发止损后又找理由扛——评估应在触发前完成
  • 用成本价/历史高点讨论止损——市场不关心你的成本
  • 信心高就重仓——仓位是风险决策不是方向决策
  • 浮盈就盲目加码——加仓要逻辑强化 + 走势验证
  • 把随机插针当破位——区分噪音与真失效
  • 形态分析取代基本面——形态只是辅助工具，基本面不过关不买
  • 形态信号不设应对方案——假突破后死扛，应在3日内认错
  • 不做风险矩阵——只有止损没有前置风险管理
一句话总结 本框架的精髓不在"找到会涨的股票"，而在"先筛掉不该碰的，再筛掉太贵的，再拆解收入质量，最后把剩下的变成有纪律的单子"。五步决策门逐级过滤，任何一步否决就不必往下——这比"分析完所有再决定"更省力也更安全。形态分析融入第五步后，"合理的估值"与"良好的技术图形"共振才是重仓出击的前提。严格执行止损纪律 + 风险矩阵前置防御 + 资金流向与形态共振确认，是把这个框架从"分析工具"变成"赚钱工具"的关键。
